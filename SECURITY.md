# Security Policy

## Supported Versions

This repository publishes container images to GHCR.

**Recommended for production:** use a pinned release tag, e.g. `ghcr.io/frepke/wetty:vX.Y.Z`.

**Supported:** the latest release (`vX.Y.Z`) and `latest` (built from `main`).

> Note: `latest` tracks `main` and may change without notice.

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security reports.

Instead, use GitHub's private vulnerability reporting:
**Security → Advisories → Report a vulnerability**.
